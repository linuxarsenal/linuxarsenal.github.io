# 进程与线程

## 进程是什么？

进程是程序的一次执行过程，是系统进行资源分配和调度的基本单位。

### 进程的组成

每个进程都由以下部分组成：
- **程序段**：存储程序的代码
- **数据段**：存储程序运行时需要的数据
- **PCB（进程控制块）**：保存进程的状态信息

### 进程的状态

```
          就绪 ──调度──► 运行
           ▲              │
           │    时间片用完  │
           └──────────────┘
                          │ 等待事件
                          ▼
                        阻塞
                          │ 事件完成
                          ▼
                         就绪
```

## 线程是什么？

线程是进程中的一个执行单元，是 CPU 调度的基本单位。

| 比较项目 | 进程 | 线程 |
|---------|------|------|
| 资源分配 | 独立分配 | 共享进程资源 |
| 切换开销 | 大 | 小 |
| 通信方式 | IPC | 共享内存 |
| 崩溃影响 | 不影响其他进程 | 影响同进程其他线程 |

## 总结

- 进程是**资源分配**的基本单位
- 线程是 **CPU 调度**的基本单位
- 一个进程可以包含多个线程，线程共享进程的地址空间
